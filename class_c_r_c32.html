Detailed Description

CRC-32.

Implementation of the 32 bit Cyclic Redundancy Check (CRC-32) algorithm as described by ISO 3309 (also see RFC 1952).

Should only be used for compatibility with legacy systems.

See also
Adler32 MD5Sum SHA1
