Detailed Description

template<class TYPE>
class ArrayOwnershipPointer< TYPE >

Array ownership automation pointer.

Automation pointer that allows transfer of ownership and automatic deletion of an array. Do not construct more than one automation pointer from the 'normal' pointer.

void MyOtherClass::myOtherMethod(MyResource resource)
{
// ...
bassert(condition, MyException(this));
// ...
}
void MyClass::myMethod()
{
ArrayOwnershipPointer<int> buffer = new int[BUFFER_SIZE];
MyOtherClass* myOtherObject = getOtherObject();
myOtherObject->myOtherMethod(resource);
int* direct = buffer.relinquishOwnership();
// ...
}
